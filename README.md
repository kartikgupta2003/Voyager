Voyager – AI-Powered Trip Planner (MERN)

• Built a responsive trip-planning platform that generates personalized travel itineraries based on user preferences, budget, and locations, reducing manual research for travelers.

• Implemented secure authentication using Google OAuth + Clerk, delivering seamless onboarding and protected access across sessions.

• Integrated MapTiler Geocoding API to fetch precise destination data and dynamically display location-based images and travel insights, enhancing user experience during trip exploration.

• Optimized backend trip CRUD operations using MongoDB, enabling fast trip creation, modification, and retrieval for returning users.

• Tech Stack used: MERN Stack, Clerk Auth, Google OAuth, MapTiler API , Google Gemini APIs , REST APIs .
